# An Agentic Automation Framework for CAD-to-CFD Workflows Agent Development

Use this skill when extending the framework modules, MCP wrappers, or workflow templates.

## Rules

- Keep private geometry out of the repository.
- Prefer high-level workflow tools over raw COM calls.
- Add JSON and Markdown reporting for new operations.
- Add unit tests for non-SolidWorks logic.
- Keep SolidWorks-dependent tests in integration folders.
- Document stop conditions and failure reports.
