# Post-processing Roadmap

Implemented first:

- Fluent report-monitor parser.
- Global pressure, temperature, velocity, species, and inlet-flow summary.
- Wall heat, wall-adjacent temperature, wall pressure, and wall-shear summary.
- Markdown and JSON report writing.
- Autosave video frame-plan generation.
- Safe postprocessing MCP inventory and wrapper.
- Public synthetic monitor example.

Next:

- Residual plotting.
- Contour and streamline export plans.
- Fluent/ParaView local renderer adapters behind explicit approval.
- Surface distribution percentiles and threshold area fractions.
- Multi-run comparison plots.
- Result package manifests for private workspaces.
