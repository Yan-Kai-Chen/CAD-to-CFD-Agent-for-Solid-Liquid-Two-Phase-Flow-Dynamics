# SolidWorks Module Overview

The SolidWorks module automates CAD operations through pywin32/COM.

Current operations include:

- preflight and connection checks,
- new part creation,
- extrusion and cutting,
- boolean combine operations,
- body transforms,
- fillet and chamfer,
- shell and thicken,
- sweep, loft, and revolve,
- STEP export and report generation.
