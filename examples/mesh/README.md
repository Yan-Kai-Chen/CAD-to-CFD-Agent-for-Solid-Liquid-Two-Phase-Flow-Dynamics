# Mesh Examples

These examples are public, synthetic, and safe to share. They do not include
private STL, CAD, Fluent, or generated runtime files.

## Examples

- `freecad_solidify/`: small ASCII cube STL for testing the FreeCAD/OpenCascade
  coarse mesh-to-solid candidate route.

Generated outputs must remain under ignored runtime folders such as
`05_projects/` or `sandbox/output/`.
