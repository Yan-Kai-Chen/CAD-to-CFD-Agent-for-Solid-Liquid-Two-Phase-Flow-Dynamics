# Public Turbulence Passport Example

This public-safe example validates a first Fluent turbulence setup for a simple
channel-like reference case. It does not run a turbulence solver.

```powershell
python -m fromcad2cfd fastcfd validate-turbulence --case-file examples\fastcfd\turbulence_channel2d_passport\turbulence_case.json --output-dir 05_projects\turbulence_channel2d_passport\reports --format json
```
